# 《开源导论与数字素养》课程指南说明

## 一、欢迎

欢迎来到《开源导论与数字素养》课程！本课程旨在作为你探索开源世界的启蒙与引导，帮助你建立开源意识、协作精神与创新思维，提升在数字时代的核心素养。无论你来自哪个专业背景，即使是零基础，这门课程都将为你打开一扇通往开源世界的大门。

---

## 二、课程简介  

课程理念：**启蒙**、**引导**、**赋能**

本课程不是一门专业的技术深潜课程，而是面向大一新生开设的通识选修课，立足于提升学生在数字时代的核心素养。通过系统学习开源的定义、历史、理念和许可证，认知全球与中国本土的开源生态，并通过真实的"微贡献"实践，完成从了解到参与的完整体验。

---

## 三、学习目标

完成本课程后，你将能够：

|目标维度|学习目标描述|
|:---|:---|
|知识层面|系统阐述开源的定义、历史、核心理念及主流许可证的差异；识别全球与中国本土的关键开源项目、基金会和平台。|
|能力层面|熟练运用Git和主流代码托管平台（如GitCode）完成版本控制与协作流程；独立查找、评估并向一个真实的开源项目提交一次"微贡献"。|
|素养层面|秉持开放、协作、共享、贡献的开源精神；建立负责任的数字公民意识；展现更高的信息素养和自主学习能力。|
|思政层面|理解开源创新与国家"技术自主可控"战略的联系，增强科技自信与社会责任感。|

## 五、教学方法

本课程采用以学生为中心的混合式学习方法：

1. **分层教学**：零基础友好—>进阶挑战—>多学科融合：鼓励跨专业组队与合作
2. **互动讲座**：结合案例故事、嘉宾分享和课堂讨论
3. **翻转课堂**：课前观看视频，课堂时间用于实践与答疑
4. **动手实验**：个人操作练习和游戏化的线上闯关活动
5. **项目式学习**：小组完成综合性期末项目
6. **同伴互评**：在代码审查等环节，学习提供和接收建设性反馈

## 十、常见问题

**Q: 我没有编程基础，能学好这门课吗？**
A: 完全可以！本课程专门为零基础同学设计，从最基础的概念和工具教起，通过丰富的非代码贡献方式参与开源。

**Q: 需要购买什么特殊软件吗？**
A: 不需要。课程中使用的所有工具和平台都是免费开源的。

**Q: 这门课的工作量大吗？**
A: 课程设计兼顾理论与实践，每周约需3-4小时。通过定期参与和循序渐进的学习，完全可以顺利完成。
